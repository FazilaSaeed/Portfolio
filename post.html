Event Management System

This project was made when I first learned java. I was the sole creator/developer of the project.

The system stored menu and other sample data for the entire event managment syatem and provided an interface for admins to view, add and send data to clients.

The final bill is generated and automatically sent to the client along with an interface for customers to be able to login and view.

Customers had the ability to view all the possible options before choosing one.

I worked on this system when I was enrolled in Object Oriented course.
